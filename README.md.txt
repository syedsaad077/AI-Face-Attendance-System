# AI Face Recognition Attendance System

An AI powered attendance system built using:

- Python
- OpenCV
- Face Recognition
- Streamlit

## Features

• Live Face Recognition  
• Add New Person  
• Attendance Dashboard  
• Date-wise Filtering  
• Delete User Option  
• Ultra Modern Glass UI  

## Tech Stack

Python, Streamlit, OpenCV, Face Recognition, Pandas, Plotly

## Run Project

Install dependencies:

pip install -r requirements.txt

Run project:

streamlit run streamlitapp.py
